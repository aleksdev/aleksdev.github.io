8!10!12!

Description

8!10!12! is a puzzle game for Android and iOS with simple but addictive gameplay. Best tetris inspired game for touch screens.

Place figures in order to create and clear full lines on the screen both vertically and horizontally, get points and unlock achievements. The game continues until you can make moves, so don't forget to keep free space for new figures.

Game field three sizes: 8x8, 10x10 and 12x12. Day and night themes. Modes without time limit and with it, no color match. Automatically saves game state on exit, you can always continue your game later.

Fourteen game modes:

  • Mini: mini figure set randomly rotated, 8x8 game field, rotation disabled;
  • Mini: mini+ figure set, 8x8 game field, rotation enabled;
  • Basic: basic figure set randomly rotated, 10x10 game field, rotation disabled;
  • Basic+: basic figure set, 10x10 game field, rotation enabled;
  • Extended: extended figure set randomly rotated, 10x10 game field, rotation disabled;
  • Extended+: extended figure set, 10x10 game field, rotation enabled;
  • Extra: extra figure set, 12x12 game field, rotation enabled;
  • Same modes but with the time limit.
