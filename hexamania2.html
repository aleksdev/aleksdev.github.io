HexaMania 2

Description

HexaMania 2 is a free puzzle game for Android and iOS.

Five interesting puzzle games in one. Despite of rules simplicity, every mode is very challenging. Everyone can find something interesting for himself.

Mode INHEX

Place shapes to the game field. Make groups of four of more tiles with the same color. These groups of tiles will be cleared and you get score points. Rotate shapes to find the best position. You can store one shape for later use and you can undo the last move. Clear the tiles to get bombs. Place the bomb to any occupied cell to clear it. The game continues as long as you can place shapes. Try to get as many points as you can.

Mode IHEX GRAVITY

Place shapes to the game field. The shapes will fall to the bottom of the game field. Make groups of four of more tiles with the same color. These groups of tiles will be cleared and you get score points. You can't rotate shapes in this game mode. You can store one shape for later use and you can undo the last move. Clear the tiles to get bombs. Place the bomb to any occupied cell to clear it. The game continues as long as you can place shapes. Try to get as many points as you can.

Mode RINGS

Place shapes to the game field. Make rings of the same color. These rings of tiles will be cleared and you get score points. Rotate shapes to find the best position. You can store one shape for later use and you can undo the last move. Clear the rings to get multicolor shapes and the bomb. Place the bomb to center of any ring to clear the tiles around it. The game continues as long as you can place shapes. Try to get as many points as you can.

Mode MERGED

Place shapes to the game field. Make groups of three of more tiles with the same number. These groups of tiles will be merged into the tile with the next number. Maximum tile number is 8. Tiles with number 8 will merge into multicolor tiles. An explosion will occur at the place where the multicolor tiles merge and clear all the tiles around. Rotate shapes to find the best position. You can store one shape for later use and you can undo the last move. Clear the tiles to get bombs. Place the bomb to any occupied cell to clear it. The game continues as long as you can place shapes. Try to get as many points as you can.

Mode MERGED GRAVITY

Place shapes to the game field. The shapes will fall to the bottom of the game field. Make groups of four of more tiles with the same number. These groups of tiles will be merged into the tile with the next number. Maximum tile number is 9. Tiles with number 9 will merge into multicolor tiles. An explosion will occur at the place where the multicolor tiles merge and clear all cells on the same row. Rotate shapes to find the best position. You can store one shape for later use and you can undo the last move. Clear the tiles to get bombs. Place the bomb to any occupied cell to clear it. The game continues as long as you can place shapes. Try to get as many points as you can.
